# TM1637_6D
Arduino library for the 6 digit TM1637 based segment displays

Wiring:</br>
Arduino Uno ------- 7-Segment module</br>
===================================</br>
5V  ---------------------> VCC</br>
GND ---------------------> GND</br>
Digital Pin 3 -----------> DIO</br>
Digital Pin 2 -----------> CLK</br>
